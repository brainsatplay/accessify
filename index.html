Enabling Web Accessibility for Low-Bandwidth Inputs

What is Accessify?

Press the tab button on your keyboard.

You'll notice that this text will get outlined.

Congratulations! You've discovered a standard accessibility feature of web browsers: you can navigate between interactable items in any webpage solely with keyboard inputs.

Additional keyboard shortcuts include:

  • Shift + Tab: Go back.
  • Arrow Up / Down: Scroll OR move between grouped items.
  • Enter / Space: Select an interactable item.
Output format

The Problem

While this navigation scheme is effective for many circumstances, it fails individuals with only a few possible inputs, such as a single switch.

Our Solution

Accessify is a tool to enable people with low-bandwidth inputs (e.g. switches, etc.) to control web browsers with faster speeds.

Using the browser as our develoment platform, we have the following benefits:

  • Written in JavaScript: Accessible to millions of developers.
  • Universal Design: Our libraries can run client and server-side (e.g. made available through a cloud API).
  • Simple Application Distribution: Anything developed alongiside our tools will have a streamlined distribution pipeline through the Web.
